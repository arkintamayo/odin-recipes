# 🍽️ Odin Recipes

## 📝 Brief Overview
This project is simple recipe website built as part of The Odin Project curriculum. The website consists of a main index page with links to a few of my personal favorite recipes.

## 🛠️ Skills Demonstrated
- HTML stcuture (headings, paragraphs, lists, links, images)
- Relative links between pages
- File organization and structure
- Git and GitHub usage

## 🧠 What I Learned
- How to build a basic multi-page website
- How to link page using relative paths
- Writing accessible content
